# Phase 1: Study — Research Template Files

Steps 1.1 and 1.2 from the [build guide](.cursor/plans/adenginebuildguide.md) (lines 119-188). Step 1.3 (calibration dataset) deferred to a separate chat.

## Step 1.1: Reference Ad Catalog

Create `[compete/references/README.md](compete/references/README.md)` — a structured template for cataloging reference ads from Slack.

Per the guide (lines 129-142), this file should prompt you to record:

- Primary text, headline, description, CTA
- Performance context (if provided)
- Personal score (1-10) on each of the 5 dimensions (clarity, value_proposition, call_to_action, brand_voice, emotional_resonance)
- What makes each ad good or bad

## Step 1.2: Competitor Patterns Tracker

Create `[compete/references/competitor_patterns.md](compete/references/competitor_patterns.md)` — a structured markdown file for documenting patterns found on facebook.com/ads/library.

Per the guide (lines 151-188), this covers:

- **Princeton Review** — hooks, CTAs, emotional angles, specific numbers/claims
- **Kaplan** — same pattern fields
- **Khan Academy** — same pattern fields
- **Chegg** — same pattern fields
- **Cross-Competitor Patterns** — most common hook type, CTA, recurring emotional angle, and gaps/opportunities for VT

Both files are templates that you'll fill in with your research. Once populated, they feed into step 1.3 (calibration dataset) in the next phase.
